    - Row layout: TextInput (flex:1) + Send button
    - Bottom padding to account for keyboard (use KeyboardAvoidingView in ChatScreen)
    - Style: dark input background matching app theme, rounded corners

    - Modal (not bottom sheet -- following project pattern of Modal for overlays)
    - Title: "Report Message"
    - Reason options as pressable rows: "Inappropriate content", "Harassment", "Spam", "Other"
    - On select: call onReport(messageId, selectedReason), then onClose
    - Cancel button at bottom
    - Do NOT add flex:1 to modal children (project memory: feedback_flex1_layout.md)

    **ChatScreen.tsx:**
    - Props: { buddyPairId: string; buddyName: string }
    - On mount (useEffect):
      1. Call messageStore.loadMessages(buddyPairId)
      2. Call flushMessageQueue() to send any queued offline messages
      3. Start Realtime subscription: const unsubscribe = messageStore.subscribeToRealtime(buddyPairId)
      4. Start message queue listener: const unsubQueue = startMessageQueueListener()
      5. Cleanup: unsubscribe() + unsubQueue() + messageStore.clearMessages()
    - FlatList with inverted={true} prop:

    - KeyboardAvoidingView wrapping the entire screen (behavior="padding" on iOS, undefined on Android)
    - ChatComposer at bottom: onSend calls messageStore.sendMessage(buddyPairId, userId, content)
      - If sendMessage returns { ok: false, reason: 'content_filtered' }: show Alert.alert('Message Filtered', 'Your message was filtered by the server. Please rephrase.')
    - Header: buddy name + three-dot menu with "Block Buddy" option

    - ReportMessageSheet: visible when selectedReportMessageId is set

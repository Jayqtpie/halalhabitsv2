    - calculateDetoxXP('deep', 8) => 300

    - calculateEarlyExitPenalty(startedAt, 4, 100, now_at_50_percent) => 50

    - calculateEarlyExitPenalty(startedAt, 4, 100, now_at_0_percent) => 100
    - isProtectedByDetox('10:00', '12:00', '09:00', '13:00') => true (habit window inside session)
    - isProtectedByDetox('10:00', '12:00', '14:00', '16:00') => false (no overlap)
    - isProtectedByDetox('20:00', '21:00', '09:00', '11:00') => false (evening habit, morning session)
    - isDeepVariantAvailableThisWeek(null) => true
    - isDeepVariantAvailableThisWeek({status: 'completed'}) => false
    - isDeepVariantAvailableThisWeek({status: 'abandoned'}) => true (re-entry allowed)
    - getRemainingMs(startedAt, 4) at midpoint => ~7200000
    - getRemainingMs(startedAt, 4) after end => 0
    - canStartSession('daily', todaySessions=[]) => true
    - canStartSession('daily', todaySessions=[{status:'abandoned'}]) => true (1 retry)
    - canStartSession('daily', todaySessions=[{status:'abandoned'},{status:'abandoned'}]) => false
    - canStartSession('daily', todaySessions=[{status:'completed'}]) => false
    - getSessionEndTime(startedAt, 4) => startedAt + 4 hours ISO string
  </behavior>
  <implementation>
    Create `src/domain/detox-engine.ts` with these exported pure functions:

    1. `calculateDetoxXP(variant: 'daily' | 'deep', durationHours: number): number`
       - Deep: always 300
       - Daily: 50 + (clampedHours - 2) * 25 where clampedHours = min(max(durationHours, 2), 6)

    2. `calculateEarlyExitPenalty(startedAt: string, durationHours: number, baseXP: number, now: string): number`
       - remainingFraction = max(0, (totalMs - elapsedMs) / totalMs)
       - return Math.round(baseXP * remainingFraction)

    3. `isProtectedByDetox(habitWindowStart: string, habitWindowEnd: string, sessionStart: string, sessionEnd: string): boolean`
       - Overlap check: hStart < sEnd && hEnd > sStart

    4. `isDeepVariantAvailableThisWeek(existingDeepSessions: Array<{status: string}>): boolean`
       - No sessions => true
       - 1 session with status 'abandoned' => true (re-entry)
       - 1 session with status 'completed' or 'active' => false
       - 2+ sessions => false

    5. `getRemainingMs(startedAt: string, durationHours: number): number`
       - endMs = new Date(startedAt).getTime() + durationHours * 3600000
       - return Math.max(0, endMs - Date.now())

    6. `canStartSession(variant: 'daily' | 'deep', existingSessions: Array<{status: string}>): boolean`
       - If any session is 'active' => false
       - If any session is 'completed' => false
       - Count of sessions: 0 => true, 1 with 'abandoned' => true (retry), 2+ => false
       - For deep: same logic but scoped to weekly (caller passes weekly sessions)

    7. `getSessionEndTime(startedAt: string, durationHours: number): string`
       - return new Date(new Date(startedAt).getTime() + durationHours * 3600000).toISOString()

    All functions: zero React imports, zero DB imports. Pure TypeScript only.
  </implementation>
</feature>

       - `titles`: id, name (unique), rarity, unlock_type, unlock_value, unlock_habit_type, flavor_text, sort_order
       - `user_titles`: id, user_id (FK), title_id (FK), earned_at. UNIQUE on (user_id, title_id)
       - `quests`: id, user_id (FK), type, description, xp_reward, target_type, target_value, target_habit_id (FK nullable), progress (default 0), status (default 'available'), expires_at, completed_at, created_at

       - `niyyah`: id, user_id (FK, unique), text, created_at
       - `sync_queue`: id, entity_type, entity_id, operation, payload, created_at, synced_at
       - `_zustand_store`: key (text PK), value (text) — for Zustand persist storage adapter

       Use `real` for REAL columns (multiplier, lat/lng).

**MuhasabahModal.tsx:**
Container component that mounts as an absoluteFillObject overlay. Reads `muhasabahStore.isOpen` and `muhasabahStore.currentStep` to determine which screen to show.

```typescript
export function MuhasabahModal() {
  const { isOpen, currentStep } = useMuhasabahStore(useShallow(s => ({ isOpen: s.isOpen, currentStep: s.currentStep })));

  if (!isOpen) return null;

  return (
    <View style={[StyleSheet.absoluteFillObject, styles.container]}>
      {currentStep === 0 && <MuhasabahStep1 />}
      {currentStep === 1 && <MuhasabahStep2 />}
      {currentStep === 2 && <MuhasabahStep3 />}
      {currentStep === 'closing' && <MuhasabahClosing />}
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    backgroundColor: 'rgba(10, 8, 28, 0.95)',
    zIndex: 100,
    justifyContent: 'center',
    alignItems: 'center',
    paddingHorizontal: 24,
  },
});
```

Add a Reanimated fade-in animation: when isOpen changes from false to true, opacity goes 0 → 1 over 300ms.

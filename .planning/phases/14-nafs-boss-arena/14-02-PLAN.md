    **Export bossRepo object with these methods:**

    **create(data: NewBossBattle): Promise<BossBattle[]>**
    - return getDb().insert(bossBattles).values(data).returning()

    **getActiveBattle(userId: string): Promise<BossBattle | null>**
    - select from bossBattles where userId = userId AND status = 'active'
    - return rows[0] ?? null

    **getLastBattle(userId: string): Promise<BossBattle | null>**
    - select from bossBattles where userId = userId AND status != 'active'
    - order by endedAt desc, limit 1
    - return rows[0] ?? null
    - Used for cooldown calculation

    **getAllBattles(userId: string): Promise<BossBattle[]>**
    - select from bossBattles where userId = userId
    - order by startedAt desc

    **updateDailyOutcome(id: string, bossHp: number, currentDay: number, dailyLog: string, updatedAt: string): Promise<BossBattle[]>**
    - update bossBattles set bossHp, currentDay, dailyLog, updatedAt where id = id
    - return .returning()

    **defeat(id: string, endedAt: string): Promise<BossBattle[]>**
    - update bossBattles set status = 'defeated', endedAt = endedAt, updatedAt = endedAt where id = id
    - return .returning()

    **escape(id: string, endedAt: string): Promise<BossBattle[]>**
    - update bossBattles set status = 'escaped', endedAt = endedAt, updatedAt = endedAt where id = id
    - return .returning()

    **getDefeatedCount(userId: string): Promise<number>**
    - select count from bossBattles where userId = userId AND status = 'defeated'
    - return count value as number

    **getDefeatedArchetypes(userId: string): Promise<string[]>**
    - select distinct archetype from bossBattles where userId = userId AND status = 'defeated'
    - return array of archetype strings

    **src/components/arena/BossHpBar.tsx:**

    Skia-drawn component rendered INSIDE the parent Skia Canvas (not its own Canvas).

    Props: `{ hpRatio: number; canvasWidth: number; y?: number }`

    Implementation per UI-SPEC BossHpBar section:
    - Track width: canvasWidth - 64 (32px padding each side)
    - Track height: 16px
    - Track x: 32, y: props.y ?? 16
    - Background track: RoundedRect, fill gold-500 (#FFD700) at opacity 0.25, r=4
    - HP fill: RoundedRect, fill ruby-600 (#DC2626), width = animated shared value
    - Animation: useSharedValue for hpWidth, useEffect watches hpRatio, drives withTiming(hpRatio * trackWidth, { duration: 600, easing: Easing.out(Easing.ease) })
    - Use useDerivedValue to compute the fill width for Skia:
      ```typescript
      const animatedHp = useSharedValue(hpRatio);
      useEffect(() => {
        animatedHp.value = withTiming(hpRatio, { duration: 600, easing: Easing.out(Easing.ease) });
      }, [hpRatio]);
      const fillWidth = useDerivedValue(() => animatedHp.value * trackWidth);
      ```
    - Render order: background RoundedRect first, then fill RoundedRect on top
    - Healing flash: optional — withSequence for brief gold flash overlay (150ms in/out per motion contract)
    - Accessibility: handled by parent ArenaScreen via accessibilityValue on a wrapping View

    Typewriter animation (setInterval pattern — NOT Reanimated):
    ```typescript
    const [displayedText, setDisplayedText] = useState('');
    useEffect(() => {
      if (!isActive) { setDisplayedText(text); return; }
      setDisplayedText('');
      let i = 0;
      const interval = setInterval(() => {
        setDisplayedText(text.slice(0, i + 1));
        i++;
        if (i >= text.length) clearInterval(interval);
      }, 30); // 30ms per character per UI-SPEC motion contract
      return () => clearInterval(interval);
    }, [text, isActive]);
    ```

    Reduced motion: check AccessibilityInfo.isReduceMotionEnabled. If true, show full text instantly (no setInterval).

    When no text/not active: show empty box with "..." placeholder in same font.

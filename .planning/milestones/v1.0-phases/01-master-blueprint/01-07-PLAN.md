**blueprint/15-content-pack.md** (BLUE-15):

Per user decision: "Content pack needs actual copy strings: 40 microcopy, 20 quest lines, 20 boss encounters, 20 mercy mode messages, 10 Friday power-ups, 20 notification templates."

All copy MUST:
- Use wise mentor voice ("Your discipline grows stronger") per locked decision
- Include Arabic terms with inline context ("Complete your Dhikr (remembrance)") per locked decision
- Pass adab safety rails: no shame, no spiritual judgment, no guilt language
- Be final-draft quality -- ready to drop into code, no placeholders

**Adab Copy Guide (write this FIRST as reference for all copy below):**
Create a do/don't guide with 10+ paired examples:
- DO: "Your momentum is building" / DON'T: "You're falling behind"
- DO: "Recovery quest available" / DON'T: "You failed your streak"
- DO: "Your discipline grows stronger" / DON'T: "Your iman score increased"
- DO: "Ready for Dhuhr?" / DON'T: "You missed Dhuhr!"
- DO: "Continue your journey" / DON'T: "Don't give up now"
Include the Islamic ethics reasoning for each don't.

**Section 1: Microcopy (40 strings)**
Organized by location:
- Buttons (8): "Complete", "Begin Recovery", "Set Niyyah", "View Quest", etc.
- Empty states (8): empty habits list, no quests available, no titles earned, etc.
- Success messages (8): habit completed, quest done, streak milestone, etc.
- Error messages (8): network error, save failed, invalid input, etc.
- Onboarding (8): welcome, niyyah prompt, habit selection, tutorial tips, etc.

For each: `[location]: "[exact copy string]"`

**Section 2: Quest Lines (20 strings)**
Each quest has: Quest Name | Description | Completion Message
Mix of:
- Daily quests (8): "Complete all 5 salah today", "Read Quran for 10 minutes", etc.
- Weekly quests (7): "Maintain a 7-day streak on any habit", "Complete 30 habit check-ins this week", etc.
- Stretch quests (5): "Complete 50 check-ins this week", "Maintain streaks on all habits for 14 days", etc.

Use templates: [verb] + [quantity] + [habit type] + [time frame]
Completion messages use celebration tone: "Quest conquered. Your discipline is undeniable."

**Section 3: Boss Encounter Messages (20 strings)**
These are v2 content but needed for economy coherence. Mark as "v2 -- pre-written for future Nafs Boss Arena."
5 boss archetypes x 4 messages each (intro, taunt, player-winning, defeated):
- The Procrastinator (nafs al-ammara -- the commanding self)
- The Distractor (scattered attention)
- The Doubter (waswasa -- whispers of doubt)
- The Comparer (comparing to others' worship)
- The Perfectionist (all-or-nothing thinking)

Boss messages use the enemy's "voice" in contrast to the wise mentor. Keep tone respectful -- these represent internal struggles, not external enemies.

**Section 4: Mercy Mode Messages (20 strings)**
Per user decision: "Mercy Mode uses blended tone: brief Islamic wisdom + game recovery action."
Each message has two parts: [Islamic wisdom line] + [Game action line]
Examples:
- "The door of tawbah (repentance) is always open. Recovery quest unlocked -- rebuild your momentum."
- "Every sunrise is a new beginning. Complete [habit] twice in the next 2 days to restore your streak."

20 unique messages ensuring variety when Mercy Mode activates. Include ayah or hadith references where appropriate (with source citations: surah:ayah or hadith collection).

**Section 5: Friday Power-Up Messages (10 strings)**
V2 content. Mark as "v2 -- pre-written for future Jumu'ah bonuses."
Messages for Friday-specific XP multiplier activation:
- "Jumu'ah blessings activated. All completions earn 2x XP today."
- Include one relevant hadith about Jumu'ah (with source)

**Section 6: Notification Templates (20 strings)**
Per user decision: "All notification copy is invitational, never guilt-based."
Categories:
- Prayer reminders (5): one per salah, using "Ready for [prayer]?" format
- Habit reminders (5): gentle, time-contextual ("Your evening Quran session awaits")
- Muhasabah prompt (3): evening reflection invitation
- Streak celebrations (3): milestone notifications
- Quest updates (2): new quests available, quest expiring
- Level up (2): congratulations, new unlock available

For each: `[type]: "[notification title]" / "[notification body]"`
